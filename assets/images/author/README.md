# Author Assets

This directory contains author-related images and assets used across multiple articles.

## Files

- `avatar.png` - Author avatar image (William J. Saraiva)

## Usage

Reference the avatar in article front matter or markdown:

```markdown
![Author Avatar](/assets/images/author/avatar.png)
```

Or in front matter for consistent author display:

```yaml
author_avatar: "/assets/images/author/avatar.png"
```
